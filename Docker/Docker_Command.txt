1. Install Docker (Docker Web site : https://www.docker.com/get-started/)
    A. macOS : type (Apple Chip or Intel Chip)
    B. windows : type (x86,x64)
    C. Linux (Ubuntu)
        --- Preparation for installation ---
        a. 'apt' Update : 
            sudo apt-get Update
        b. Using repositories via HTTP : 
            sudo apt-get install -y \ apt-transport-https \ ca-certificates \ curl\ software-properties-common
        c. Add GPG Key : 
            curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
        d. Check GPG Key : 
            sudo apt-key fingerprint OEBFCD88
        e. Register Rpository : 
            sudo add-apt-repository \ "deb [arch=amd64] https://download.docker.com/linux/ubuntu \ $(lsb_release -cs) \ stable"
            sudo apt-get Update
        --- Install Docker
        f. sudo apt-get install docker-ce

2. Docker Coommand 
    A. Check Docker Version
        docker Version
    B. Check Execution Environment
        docker system info
    C. Docker use Disk Status
        docker sytem df
    D. Check Dokcer Images
        docker image ls
    E. Docker Image Download (use Docker Hub)
        docker pull [docker-image-name]
    F. Docker Container Status
        docker container ps
    G. Container Running Status
        docker container stats [container-name]
    H. Container Stop 
        docker stop [container-name]
    I. Container Start
        docker start [container-name]

3. Docker Run Web Server(Nginx)
    A. Docker Image Download
        docker pull nginx
    B. Run Docker Image
        docker container run --name webserver -d -p 80:80 nginx
